Sobre o Projeto On Class

O On Class é um sistema inovador desenvolvido para revolucionar o controle de acesso em instituições de ensino, utilizando tecnologia de QR Code para garantir segurança, eficiência e transparência.

Problema Identificado

Instituições escolares enfrentam desafios no gerenciamento de alunos, com falta de registro claro de entrada e saída, dificuldade em fornecer informações aos responsáveis e ausência de autenticação eficiente de identidade.

Solução Proposta

Implementação de um sistema completo com crachás de QR Code único para cada aluno, leitura rápida na portaria, geração de relatórios automáticos e acesso remoto para responsáveis.

Funcionamento
  1. Cadastro do aluno no sistema pelo administrador
  2. Geração de crachá com QR Code único
  3. Leitura do QR Code na portaria para registro de entrada/saída
  4. Notificação automática para responsáveis
  5. Geração de relatórios para a equipe administrativa

Histórias de Usuário

US01 - Porteiro

Como porteiro, quero ler o QR Code do crachá do aluno para registrar sua entrada na escola.

US02 - Porteiro

Como porteiro, quero confirmar a saída do aluno para manter o controle de quem ainda está no campus.

US03 - Aluno

Como aluno, quero usar meu crachá com QR Code para registrar minha presença.

US04 - Responsável

Como responsável, quero ver os horários de entrada e saída do meu filho para garantir sua segurança.

US05 - Direção

Como diretor, quero emitir relatórios de frequência para apoiar o planejamento e gestão escolar.

US06 - Administrador

Como administrador, quero cadastrar alunos no sistema para gerar seus crachás com QR Code.
